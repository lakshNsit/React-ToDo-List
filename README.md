# React-ToDo-List
Minimalistic React ToDo List
How To Use?
Download the source code
Unzip the folder
Open terminal at the folder.
Run 'npm install'
Run 'npm start'
And the To-Do List is ready to be used to organize and prioritize your daily tasks.
