# React-ToDo-List
Minimalistic React ToDo List

How To Use?

1) Download the source code

2) Unzip the folder

3) Open terminal at the folder.

4) Run 'npm install'

5) Run 'npm start'
And the To-Do List is ready to be used to organize and prioritize your daily tasks.
